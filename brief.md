I am building a prototype for a personalized self-discovery book called SOURCE.

Users will answer a guided questionnaire (approximately 50–80 questions) using either text or voice input. Their answers will then be used to generate a personalized book based on their own life experiences. The book content will be generated using the reader's questionnaire responses as examples, reflections, and case studies throughout the chapters.

This is not a storytelling app, children's book, or choose-your-own-adventure experience. The reader's own life becomes the content of the book.

For Version 1, I am primarily interested in validating the core user experience. The prototype should include:

• Landing page
• User accounts and login
• Questionnaire with text and voice input
• Save progress and return later
• AI-generated personalized book
• Mobile-friendly design
• Simple admin dashboard

I am open to no-code, low-code, or custom-coded solutions. My goal is to build the simplest working proof of concept that can be tested with real users and expanded later if validated.
